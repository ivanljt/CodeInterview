## 赛马问题

25匹马、5条赛道、选出速度最快的三匹马至少需要多少次？

``` 
A 1，2，3，4，5
B 1，2，3，4，5
C 1，2，3，4，5
D 1，2，3，4，5
E 1，2，3，4，5

①每一组排除掉第 4，5 名

A 1，2，3
B 1，2，3
C 1，2，3
D 1，2，3
E 1，2，3

②每一组的第一名 出来比赛（这一轮可以确定第一名）。假设 A B C 里面的第一名分别是 第一第二第三名，那么 D E 两组的人可以完全剔除掉。C 组中的 2，3 名 也是。B 组里面的第三名也是。
A 1，2，3
B 1，2
C 1，

③从以下5名中选出第二第三名
A 2，3
B 1，2
C 1，


所以，总共是 5 + 1 + 1 = 7次。
关键是第②步，充分利用了每组组内的排名优势
```
